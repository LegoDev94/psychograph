# Психограф — онлайн-тестирование по методике ММИЛ

Демонстрационная реализация сайта психологического онлайн-тестирования по ТЗ:
методика ММИЛ (адаптация MMPI Ф. Б. Березина и М. П. Мирошникова), бесплатный
график-профиль в T-баллах и платная ИИ-интерпретация (DeepSeek).

**Живая версия:** https://legodev94.github.io/psychograph/

## Что реализовано (статическая версия для GitHub Pages)

| Модуль ТЗ | Реализация |
|---|---|
| M1 Главная и каталог | `index.html`, `catalog.html`, `about.html`, FAQ, правовые страницы |
| M2 Прохождение теста | выбор муж./жен. формы, согласие 18+/ПДн, «Верно / Неверно / Не знаю», прогресс, автосохранение, пауза/продолжение, навигация назад-вперёд, клавиатура |
| M3 Расчёт и профиль | сырые баллы по ключам, K-коррекция (Hs +0.5K, Pd +0.4K, Pt +1.0K, Sc +1.0K, Ma +0.2K), T = 50 + 10(X−M)/σ по нормам пола, оценка достоверности L/F/K и F−K, график ECharts с коридором нормы 30–70, экспорт PNG/PDF |
| M4 ИИ-анализ | опросник из 15 вопросов, обезличивание данных, демо-движок интерпретаций **или** живой вызов DeepSeek API со своим ключом, отчёт с дисклеймером |
| M5 Оплата | имитация сценария эквайринга (заказ → оплата → callback → доступ), идемпотентность «одна оплата — одна генерация» |
| M6 Личный кабинет | история результатов в localStorage, экспорт JSON, право на удаление |
| M9 Уведомления | тосты о событиях (email-часть требует сервера) |

Серверные части ТЗ (Laravel 11 + Filament, MySQL, Redis-очереди, реальный
эквайринг Альфа-Банка, 54-ФЗ, полнотекстовый поиск, админ-конструктор тестов)
на GitHub Pages выполняться не могут — для них в ТЗ описан отдельный стек.
Данная статика полностью повторяет пользовательские сценарии и переносится
на Laravel-бэкенд заменой слоя хранения.

## Важно: демо-банк утверждений

Оригинальные 377 утверждений ММИЛ, ключи и нормы — лицензируемый методический
материал. В репозитории используется **демонстрационный банк из 102 оригинальных
утверждений** и условные нормы, сгенерированные для проверки движка.
Владелец прав на методику подключает полный банк заменой
`assets/data/mmil.js` (структура: `items[] { id, text, keys[] {scale, keyed, keyedFemale?} }`,
`norms {male|female} {scale: {m, sd}}`) — движок расчёта менять не нужно.

## Технологии

- Ванильный HTML/CSS/JS (ES-модули), без сборки — деплой строкой на Pages
- [Apache ECharts](https://echarts.apache.org/) — график-профиль
- Дизайн-система «научная монография»: STIX Two Text / IBM Plex Sans / IBM Plex Mono, светлая и тёмная темы
- Палитра графиков валидирована на цветослепоту и контраст

## Локальный запуск и тесты

```bash
# любой статический сервер, например:
npx serve .
# тесты движка расчёта:
node tests/engine.test.mjs
```

## Структура

```
index.html              главная (герой с живым профилем, шаги, тарифы, FAQ)
catalog.html            каталог тестов
test.html               прохождение теста (3 экрана)
results.html            профиль: график, таблица шкал, достоверность, экспорт
analysis.html           ИИ-интерпретация: опросник → оплата (демо) → генерация
history.html            история результатов (локально)
about.html              о методике и проекте
privacy/offer/terms     правовые шаблоны (152-ФЗ / 54-ФЗ, требуют юриста)
assets/js/engine.js     движок расчёта (чистые функции, покрыт тестами)
assets/js/profile-chart.js  спецификация графика-профиля
assets/data/mmil.js     определение теста: пункты, ключи, нормы, правила
assets/data/interp.js   библиотека интерпретаций демо-движка
tools/integrate.mjs     генератор данных/вставка контента
tests/engine.test.mjs   юнит-тесты формул
```
